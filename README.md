# mhborthwick.me

> Personal website built with Astro and React.

## Requirements

- Node.js `24.12.0`
- pnpm `10.27.0`

## Setup

```bash
pnpm install
```

## Development

```bash
# Start development server
pnpm dev

# Build for production
pnpm build

# Preview production build
pnpm preview
```

## Testing

```bash
# Run tests
pnpm test

# Run tests with coverage
pnpm coverage
```

## Code Quality

```bash
# Format and lint code
pnpm check

# Format only
pnpm format

# Lint only
pnpm lint

# CI check (no write)
pnpm ci
```

## Technologies

- **Astro** - Static site framework
- **React** - UI components
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Biome** - Linting and formatting
- **Vitest** - Testing framework
- **React Testing Library** - Component testing

## Deployment

Deployed to [Cloudflare Pages](https://pages.cloudflare.com/).
